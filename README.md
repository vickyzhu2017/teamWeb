#团队博客管理后台(teamWeb)
-   后台：nodejs express
-   数据库：mongodb
-   html模板引擎：jade
-   js框架：jquery
-   css预处理语言：less
-   基于npm scripts来集成开发命令

## 工程目录
    - assets/                       # 存放项目资源
    - common/                       # 存放跟业务相关的公共模块
    - controllers/                  # 存放业务逻辑相关
    - node_modules/                 # 存放三方包
    - data/                         # 存放mongodb数据文件
    - test/                         # 临时后台视图文件
    - router.js                     # 路由入口
    package.json                    # 元数据文件
    app.js                          # 启动nodejs服务
    README.md                       # 说明文件


## how to use
```
git clone https://github.com/vickyzhu2017/teamWeb.git
cd teamWeb && npm install
npm run start
```
